# E-Commerce Sales Analysis | EDA + RFM Segmentation

Comprehensive Exploratory Data Analysis on Online Retail dataset with Customer Segmentation.

## 📊 Project Overview
- **Dataset**: UK-based Online Retail transactional data
- **Technologies**: Python, Pandas, NumPy, Matplotlib, Seaborn
- **Key Analysis**: Data Cleaning, Revenue Trends, Top Products, Regional Performance, RFM Segmentation

## ✨ Key Features
- Full data cleaning and preprocessing
- Revenue analysis and seasonality trends
- Top 10 products and countries
- RFM Analysis with 4 customer segments (Champions, Loyal, At-Risk, Lost)
- Professional visualizations (10+ plots)
